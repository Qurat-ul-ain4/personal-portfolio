# 🌐 Personal Portfolio Website

A premium, dark-themed responsive portfolio website built from scratch using **HTML5** and **CSS3**. 

This website serves as my digital resume to host my academic background, technical skills, and professional experiences in one place.

## 🎯 Key Features
*   **Dual-Profile Showcase:** Highlights both my Computer Science engineering background and my 1+ year experience in Digital Marketing & Content Writing.
